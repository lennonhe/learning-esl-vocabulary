## Guidelines for Common Patterns

The "Common patterns to remember" section should:
- Be based strictly on real, everyday usage patterns
- Include 2-4 common collocations, sentence structures, or fixed expressions
- Use quotation marks around the pattern itself
- Provide a brief explanation after the em dash (â€”)

**Do NOT make up patterns.** Only include patterns that are genuinely common in daily English usage.

> For more detailed guidelines, see [writing-guidelines.md](writing-guidelines.md)

## Formatting Guidelines

- Use lowercase for the title (word/phrase) - do not capitalize
- Include pronunciation with both British (BrE) and American (AmE) IPA
- Bold the "Date learned" label
- Use "### Meaning 1", "### Meaning 2", etc. for all entries (even single-meaning words)
- Include part of speech in bold before the definition
- Number example sentences
- Include "Typical usage" for each meaning
- Include "Common patterns to remember" for each meaning (based on real daily usage only)
- Use horizontal rules `---` to separate meanings from memory tips
- Number memory tips
- Keep formatting consistent across all entries

## Guidelines for All Entries

- Use lowercase for the title (word/phrase) - do not capitalize
- Always include pronunciation with both British and American IPA
- Use the same format for single-meaning and multiple-meaning words
- For multiple meanings, you MUST mark the most common meaning (based on real-world usage frequency) with "â˜… Most common" on the meaning header. Order meanings from most to least common. See [multi-meaning-template.md](multi-meaning-template.md)
- Include the part of speech for each meaning (it may differ between meanings)
- Provide 2-4 example sentences per meaning
- Add "Typical usage" to help learners know when to use each meaning
- Add "Common patterns to remember" with 2-4 common collocations or sentence structures (strictly based on real daily usage)
- Use horizontal rules `---` to separate meanings visually
- Memory tips should help remember the word (for single meaning) or connect/distinguish between meanings (for multiple meanings)
- If pronunciation varies by part of speech, indicate this clearly in the pronunciation section
